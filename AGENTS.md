# AGENTS.md

## Cursor Cloud specific instructions

This is a content-only repository containing a curated "Awesome Stars" list (`README.md`) auto-generated by the [starred](https://github.com/maguowei/starred) Python tool. There are no applications, services, tests, or build systems.

### Tools

- **starred** (Python): Generates the `README.md` from a GitHub user's starred repos. Requires `--username` and `--token` flags. Install with `pip3 install starred`. The `starred` binary installs to `~/.local/bin` — ensure that is on `PATH`.
- **markdownlint-cli** (Node.js): Lints `README.md`. Install globally with `npm install -g markdownlint-cli`. Note: lint warnings are expected because the markdown is auto-generated by `starred` and not hand-authored.

### Common commands

| Task | Command |
|------|---------|
| Lint | `markdownlint README.md` |
| Regenerate list | `starred --username linweisong --token "$GITHUB_TOKEN" --sort > README.md` |
| Check starred version | `starred --version` |

### Notes

- The `gh` CLI is pre-authenticated; use `gh auth token` to get a GitHub token for `starred`.
- `starred` outputs to stdout when no `--repository` is specified; redirect to `README.md` to update the file.
- Markdownlint will report many warnings on auto-generated content (line length, trailing spaces, indentation). These are expected and do not indicate a problem.
