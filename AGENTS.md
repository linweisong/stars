# AGENTS.md

## Cursor Cloud specific instructions

This repository is a static "Awesome List" — a curated collection of GitHub starred projects stored entirely in `README.md`. It was generated by the [starred](https://github.com/maguowei/starred) Python tool. There is no source code, build system, or test suite.

### Development tools

- **Lint**: `markdownlint README.md` (installed globally via npm). Reports style issues but the existing README has known lint warnings (line length, trailing spaces, indentation) that are artifacts of the `starred` generator.
- **Preview**: `grip README.md 0.0.0.0:6419` renders the README with GitHub-flavored styling at `http://localhost:6419`. Grip is installed via pip. Note: grip downloads GitHub CSS on first run, so first launch takes a few seconds.

### Caveats

- The repo has no automated tests, no CI, and no build step.
- Editing workflow is: edit `README.md` → lint → preview → commit.
